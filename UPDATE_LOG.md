# Update Log

## 2026-07-12 (Bamboo Housing discovery, 2-bed pool, auto-pricing fix)
Major correction: a user found a real 4-month sublet (250 Keats Way) on bamboohousing.ca that this site's Scenario A note had claimed essentially didn't exist. Investigated why and found the real cause: Bamboo Housing is server-rendered (each page embeds full listing data in a `__NEXT_DATA__` JSON block) but a plain WebFetch of the marketing homepage strips that and returns an empty shell, so an earlier pass concluded the site had no usable inventory. It actually holds 914 active Waterloo listings, 739 of them "4 Month Sublet." Crawled all 31 pages directly with curl, found 24 clean September-start 4-month listings after screening, and added 14 of the strongest (commute, price, and especially near-whole-unit matches like 298 Hemlock St where both bedrooms of a 2-bed unit are posted together, and 1 Columbia St W where 3 of 5 rooms are posted as one group) as genuine Scenario A entries. Added 3 more per-room Scenario B listings from the same crawl. Rewrote the scenarioANote and the Bamboo Housing sourceCoverage entry to state the corrected picture plainly rather than soften it. Documented the full crawl method in SEARCH_SPEC.md so future runs don't lose this capability.

Also: added a 2-bedroom filter tab and pool (6 listings, thin like 6-bed, both sourced from this same Bamboo crawl), and replaced the manual "split rent between 5/6 people" toggle with automatic per-listing pricing, per-person rent now always divides by each listing's own bedroom count with no manual step, plus an optional "double up a room" (+1) toggle that applies universally. Site total: 47 to 64 listings (6 two-bed, 16 three-bed, 17 four-bed, 22 five-bed, plus outliers). Also caught and fixed: lastUpdated had drifted to 2026-07-07 because the scheduled daily task hadn't run in 5 days (expected behaviour, it only fires while the app is open, not a bug, documented in SEARCH_SPEC.md so it isn't mistaken for one again).

Scenario A finding: genuine confirmed whole-group 4-month (Sept-Dec 2026) leases are essentially nonexistent right now. Only one true match surfaced across both passes, a 2-bedroom apartment sublet, too small to qualify. Rather than pad the list with fabricated or non-qualifying Scenario A entries, added an "askAboutShortTerm" flag to the 10 strongest Scenario B listings most likely to have a flexible landlord, surfaced as a dedicated "Worth a Call About a 4-Month Term" section on the site.
