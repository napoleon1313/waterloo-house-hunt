# Update Log

## 2026-07-07 (retro-futuristic redesign)
Restyled both pages in the manner of kanye2049.com, the retro Mac System OS concept site: black CRT frame with scanlines, subtle flicker, and phosphor glow around a dithered grey desktop; every listing card is now a System-OS window with a pinstriped title bar (the address is the window title and the link); pixel fonts (VT323, Silkscreen) for chrome and Space Mono for data; a sticky menu bar with Current/Archive/Source and a live clock; and a one-click boot screen on first visit per session ("WATERLOO HOUSE HUNT OS v2.049", no password). Money values render in terminal green. Design tokens were taken from an open-source reimplementation of the reference site (Chicago-style bitmap type, rgb(44,44,44) borders, the exact classic Mac dither pattern, red accents). All functionality is unchanged: bedroom filter, rent splitter, tiers, archive immutability. Reduced-motion preferences disable the flicker and blink effects. No data changes this pass.

## 2026-07-07 (weekly archive added)
Built a permanent historical view. Every ISO week, the current listings.json is frozen into `archive/<isoWeek>.json` and indexed in `archive/index.json`; once written, an archived week is never edited again even as the live site keeps changing. Added `history.html`, a public page (linked from the main site's header) that lists every archived week with that week's top picks shown by default and the rest expandable on click. Extracted the shared styling and card-rendering logic into `style.css` and `site.js` so both pages stay visually and behaviourally consistent. Seeded the first snapshot now as "Week of July 6, 2026" (2026-W28) using the current 47-listing dataset, since today falls partway through the week; going forward, the daily scheduled task creates a new snapshot automatically on the first run of each new ISO week and otherwise leaves the archive untouched.

Scenario A finding: genuine confirmed whole-group 4-month (Sept-Dec 2026) leases are essentially nonexistent right now. Only one true match surfaced across both passes, a 2-bedroom apartment sublet, too small to qualify. This tracks with how Waterloo's co-op sublet market actually works: departing students sublet individual rooms, not whole houses as a group, and most fall-term postings do not appear until August. Rather than pad the list with fabricated or non-qualifying Scenario A entries, added an "askAboutShortTerm" flag to the 10 strongest Scenario B listings most likely to have a flexible landlord, surfaced as a dedicated "Worth a Call About a 4-Month Term" section on the site. Excluded from the main list: a Cardill Cres listing with inconsistent per-room pricing across ads (soft scam-adjacent signal), a stale "no vacancy" Albert St listing, and a couple of unverifiable leads.
